# Docker Control Center v1.1.0

## EN
This update improves first-run and restart experience for local Docker users.

### Highlights
- New `Configure -> Application` panel.
- New option: start Docker Desktop in the background when DCC starts (local mode).
- Improved local Docker error dialog with one-click Docker Desktop start.
- Full EN/PL localization in New Container catalog descriptions/notes.
- Safer and clearer startup flow after Windows reboot.

## PL
Ta wersja poprawia wygodę pracy lokalnej po starcie systemu.

### Najwazniejsze zmiany
- Nowy panel `Konfiguruj -> Aplikacja`.
- Nowa opcja: uruchamianie Docker Desktop w tle przy starcie DCC (tryb lokalny).
- Lepszy komunikat braku lokalnego Dockera z przyciskiem uruchomienia Docker Desktop.
- Pelna lokalizacja EN/PL opisow i podpowiedzi w katalogu Nowy kontener.
- Czytelniejszy i bezpieczniejszy flow startu po restarcie Windows.
